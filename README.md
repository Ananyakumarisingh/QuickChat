## Run Locally

Clone the project

```bash
  git clone https://github.com/Ananyakumarisingh/mern-chat-app
```

Go to the project directory

```bash
  cd QuickChat
```

Install dependencies

```bash
  npm install
```

```bash
  cd frontend/
  npm install
```

Start the server

```bash
  npm start
```
Start the Client

```bash
  //open now terminal
  cd frontend
  npm start
```
